# Rock Paper Scissors Game (Python)

## 🎮 About the Project:-

This is a simple **Rock, Paper, Scissors** game made using Python. The player competes against the computer, which makes a random choice each round.

This project is beginner-friendly and is a great way to learn basic Python concepts such as:

* Variables
* Lists
* User input
* Conditional statements (`if`, `elif`, `else`)
* Random module

## 🚀 Features:-

* Play Rock, Paper, or Scissors against the computer.
* Computer chooses randomly.
* Checks for valid user input.
* Displays whether you **win**, **lose**, or **draw**.
* Easy to understand and modify.

## 🛠️ Technologies Used

* Python 3
* `random` module

## ▶️ How to Run

1. Download or clone this project.
2. Open the project folder.
3. Open a terminal in the project folder.
4. Run the program:
   
   python rock_paper_scissors.py

6. Enter one of the following choices when prompted:

   * rock
   * paper
   * scissors

## 📸 Example Output

```text
Enter rock, paper, or scissors: rock
Computer chose: scissors
You win!
```
